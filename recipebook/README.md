## **Site Navigation**

1. **Homepage**:
   - Visit the homepage to see an overview of all available recipes. Use the sidebar to quickly navigate to other sections of the app.

2. **Viewing Recipes**:
   - Click on any recipe in the **Items List** to view its details on a dedicated **Item Details Page**.

3. **Adding New Recipes**:
   - Navigate to the **Create Item Form** using the sidebar or directly from the homepage, allowing you to add a new recipe to the collection.

4. **Editing Recipes**:
   - On each recipe's details page, there is an option to update or edit the recipe information via the **Update Item Form**.

5. **Deleting Recipes**:
   - Each item in the Items List has a delete button, enabling quick removal of recipes directly from the list.

6. **About Page**:
   - Visit the **About Page** to learn more about the project and its contributors. Links to each contributor's GitHub and LinkedIn profiles are included here for networking and collaboration.

7. **Not Found Page**:
   - If you navigate to an invalid route, a custom **Not Found Page** will guide you back to the main site sections.

## **Getting Started**

### **Installation**
To set up the project locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/recipebook.git
   ```
2. Navigate into the project directory:
   ```bash
   cd recipebook
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:3000) to view the app in the browser.

### **Building for Production**
Create an optimized build:
```bash
npm run build
```
